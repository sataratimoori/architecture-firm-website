# Architecture Firm Website

A modern and responsive architecture firm website built as a collaborative team project. This website showcases architectural services, projects, team members, and contact information with a clean and elegant user interface.

## Features

- Responsive Design
- Modern User Interface
- Architecture Project Showcase
- Team Section
- Contact Form
- Smooth Animations

## Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
